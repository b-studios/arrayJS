    arrayjs.Collection

    For information on how to retreive a Collection-Object from your list of elements see the factory method.

    All default Array-Operations can be used also on a arrayJS-Collection. Implemented in Array by default:

    • push
    • pop
    • shift
    • unshift
    • slice
    • splice
    • sort
    • filter (reimplemented this one for consistency)

    Most of the operations found below can be chained, like:

    var myArray = _([10, 2, 4, 8]);  

myArray.filter(function() { return this > 3; })  
       .collect(function() { return this*2;})  
       .each(function(i) { console.log(this, i); });  

> 20 0
> 8 1
> 16 2
> [20, 8, 16]
